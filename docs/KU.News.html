Namespace: News

KU. News

Contains all news related functions for loading and controlling the news page.

Members

<static> listQueue :Array.<Object>

Contains the current list of article DOM li tag items that still need to be added to the DOM (much faster to add all at once after load is done downloading). This helps prevent the app from seeming to hang or become unresponsive.
Type:
  • Array.<Object>

<static, constant> LOAD_THRESHOLD_PX :int

Designates the minimum number of pixels that the user can scroll (calculated from the bottom) before another load event is triggered.
Type:
  • int

<static> loading :boolean

Tells whether or not news is currently attempting to download or parse article lists. Essentially used to tell whether news is considered to be busy.
Type:
  • boolean

<static> page :int

Stores the last page number downloaded as denoted from Kettering's news website kettering.edu/news/current-news. Indexing starts off at 0 and continues on to an unbounded limit.
Type:
  • int

<static> pageQueue :Array.<Object>

Contains the current list of pages DOM data-role="page" items that still need to be added to the DOM (much faster to add all at once after load is done downloading). This helps prevent the app from seeming to hang or become unresponsive.
Type:
  • Array.<Object>

<static> queue :int

Number of pages that need to be downloaded still. This is used to dynamically and asynchronously download more than one page at a time, which is configured at KU.PAGES_TO_LOAD
Type:
  • int

Methods

<static> downloadArticle(domId, link)

Downloads and parses a Kettering news article in order to display detailed article to the user. Usually this is triggered by a user clicking on an article link from the main news page.
Parameters:
Name Type Description
domId string The article DOM identifier for the link that was clicked by the user (e.g in html format, id="News-1-0") which represents the first page article 0.
link string Directly link to the news article located on Kettering's main website.

<static> getNextPage()

Downloads the next page in order to gain another 10 articles as part of the article list for the main news page. Note that there are no arguments, but the function uses namespace members such as KU.News.page and KU.News.queue.
