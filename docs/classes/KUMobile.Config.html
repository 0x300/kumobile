KUMobile.Config

Module: KUMobile

Contains configuration settings for the KUMobile project

  • Index
  • Attributes

Attributes

  • DEFAULT_INCR_WAIT_TIME
  • isAndroid
  • isDevice
  • isIOS
  • isWindows
  • LOAD_INDICATOR_DELAY

DEFAULT_INCR_WAIT_TIME

int

Default wait time (ms) for incremental searching.

Default: 800

isAndroid

boolean

Quick access boolean to determine if the device is of Android platform.

Default: false

isDevice

boolean

Is this a device (otherwise assume its a browser/emulator). Some plugins cannot be used in a browser/emulator and thereby must be determined during runtime.

Default: false

isIOS

boolean

Quick access boolean to determine if the device is of iOs platform.

Default: false

isWindows

boolean

Quick access boolean to determine if the device is of Windows platform.

Default: false

LOAD_INDICATOR_DELAY

int

Delay for loading minimum time (ms) for loading indicator to show for. This was mainly useful to prevent the indicator confusingly flashing.

Default: 400
